Smart Energy Saving System

Project Overview

The Smart Energy Saving System is an automation-based solution that reduces electricity wastage using motion sensors. The system automatically controls lights and fans based on human presence in a room, improving energy efficiency and reducing power consumption.

Problem Statement

In homes, classrooms, offices, and public spaces, electrical appliances are often left ON unnecessarily, leading to energy wastage and increased electricity bills.

Objectives

- Reduce electricity wastage
- Automate control of lights and fans
- Improve energy efficiency
- Lower electricity costs
- Promote sustainable energy usage

Working Principle

1. Motion sensors detect human presence.
2. Sensor data is sent to the controller.
3. Lights and fans turn ON automatically when a person is detected.
4. Appliances turn OFF when the room becomes vacant.

Features

- Automatic light control
- Automatic fan control
- Motion detection
- Energy-saving dashboard
- User-friendly interface

Benefits

- Energy conservation
- Reduced electricity bills
- Improved efficiency
- Eco-friendly solution
- Supports sustainable development

SDG Mapping

This project supports SDG 7: Affordable and Clean Energy by improving energy efficiency and reducing unnecessary power consumption.

Technologies Used

- HTML
- CSS
- JavaScript
- Motion Sensor Concept

Conclusion

The Smart Energy Saving System is an effective solution for reducing electricity wastage through automation. It provides a simple, cost-effective, and environmentally friendly approach to energy management.
